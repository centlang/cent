# Contributing to Cent

Thanks for taking your time to contribute! It truly means a lot! ❤️

## I found a bug!

- Before reporting, make sure it's not already tracked under
  [Issues](https://github.com/centlang/cent/issues).
- If it isn't, [open a new Issue](https://github.com/centlang/cent/issues/new)
  and provide as much relevant information as possible.

## I'd like to suggest a new feature!

- [Open a new Issue](https://github.com/centlang/cent/issues/new) and describe
  why you think the addition would be useful.
- Please don't open a Pull Request before getting feedback on the feature.

## I've fixed a bug!

- Open a new Pull Request with the fix.
- If the bug is already tracked under
  [Issues](https://github.com/centlang/cent/issues), reference it.
- If it isn't, describe the problem and the proposed solution.

## I've implemented a new feature!

- Open a new Pull Request with the feature, referencing the issue number.
